# B.movies - Premium Movie Landing Page

A sleek, modern, and highly responsive movie landing page designed to show the latest cinematic experiences. This project features a high-end UI with a focus on visual impact and smooth user interactions.

## 🌟 Key Features

- **Modern Glassmorphism Design**: The header utilizes a sophisticated frosted glass effect (`backdrop-filter: blur`) for a premium aesthetic.
- **Dynamic Hero Slider**: An interactive featured movie carousel powered by **Swiper.js**.
- **Responsive Movie Grid**: A clean, organized grid of "Now Opening" movies with genre and duration details.
- **Premium Hover Interactions**: Enhanced movie card animations with smooth scaling and brightness adjustments on hover.
- **Coming Soon Section**: A dedicated category for upcoming releases with an easy-to-navigate slider.
- **Newsletter Subscription**: A stylized form for users to subscribe for latest movie updates.
- **Integrated Media**: Curated video and audio elements for a truly immersive experience.
- **Fully Responsive**: Optimized for desktops, tablets, and smartphones.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3 (Vanilla)**: Custom styling including Flexbox, Grid, and advanced animations.
- **JavaScript (Vanilla)**: For modular functionality and DOM manipulation.
- **Swiper.js**: Modern touch sliders for the hero and categories.
- **Boxicons**: High-quality vector icons for a consistent visual language.
- **Google Fonts (Poppins)**: Clean and modern typography for ultimate readability.
